## Quick Notes

- Advanced Java Q1 uses MySQL with JDBC transaction handling.
- Advanced Java Q2 is a Spring Boot REST endpoint for user registration.
- Advanced Java Q3 is a Spring Boot JPA entity that creates the `students` table in MySQL.
- Cloud Q1 and Q2 are written as design answers and can also be reviewed in the combined overview file.
- Devops contains a Jenkins pipeline that builds, tests, and archives the Spring Boot project.
- Operating System contains a C program for the non-preemptive Priority CPU scheduling algorithm.
